Intro

Sass is a CSS pre-process. It helps reduce the repetition in your code.

With use of nesting and other tools, your CSS will also be noticibly neater and readable.

A few others worth noting that you can take a look at are:

  • Less
  • CSS-Crush
  • Stylus
Variables

Sass variables are names that start with $ and are assigned a value. You can then use the name instead of the value.

Useful to keep your code DRY, do math, etc.

$my-color: #lightblue; .button { color: my-color;
} .h6 { color: my-color;
}
Nesting

If you're working with multiple divs or tags that are nested within parent tags, nesting will make your CSS once again easier to understand and read. You'll no longer need to call the class and insert its CSS. Then call it again with the element you want to change.

CSS - Here we are changing the content classes color to lightblue. However, since we want the h1's in our content class to be red. We need to recall the class with the h1 element.

.content { color: lightblue; } .content h1{ color: red; }

Sass - With Sass we no longer need to re-call the class. You can nest the h1 within it's parent (.content) to change it's content. Nesting can go as deep as you'd like.

.content{ color: lightblue; h1{ color: red; } }
